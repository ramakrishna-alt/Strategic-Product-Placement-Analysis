# Strategic Product Placement Analysis

## 📌 Overview

**Strategic Product Placement Analysis** is a Tableau-based Business Intelligence project that analyzes the relationship between product positioning, sales performance, and consumer behavior. The objective is to identify how different product placement strategies influence customer purchasing decisions and overall sales. The project provides actionable insights through interactive dashboards and visual stories, enabling retailers to optimize product placement and improve revenue.

---

## 🎯 Problem Statement

Retail businesses invest heavily in product placement strategies such as shelf positioning, promotional displays, and category organization. However, identifying which strategies actually improve sales and customer engagement can be challenging.

This project analyzes retail sales data, product placement information, and customer demographics using Tableau to determine:

* Which product positioning strategies generate the highest sales
* How customer demographics influence purchasing behavior
* Which product categories perform best in different placements
* Recommendations to improve product visibility and increase revenue

# ▶️ How to Run the Project

1. Clone the repository.

```bash
git clone https://github.com/your-username/Strategic-Product-Placement-Analysis.git
```

2. Navigate to the Flask application.

```bash
cd Flask_App
```

3. Install dependencies.

```bash
pip install -r requirements.txt
```

4. Run the Flask application.

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```
