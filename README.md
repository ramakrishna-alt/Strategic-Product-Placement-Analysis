# Product Positioning Analysis using Tableau

## 📌 Project Overview

This project analyzes the relationship between **product positioning**, **sales performance**, and **consumer behavior** using **Tableau**. The objective is to identify how different product positioning strategies influence customer purchasing decisions and overall sales performance.

By creating interactive dashboards and visualizations, this project provides actionable business insights that help organizations optimize product placement, improve marketing strategies, and increase revenue.

---

## 🎯 Problem Statement

A retail company wants to understand the impact of product positioning on its sales and consumer behavior. The company has collected data on:

- Sales performance
- Product placement
- Customer demographics
- Consumer purchasing behavior

Using Tableau, the project identifies the most effective positioning strategies and helps stakeholders make data-driven business decisions.

---

## 🚀 Objectives

- Analyze the relationship between product positioning and sales.
- Identify high-performing products and categories.
- Understand customer purchasing behavior.
- Compare sales performance across different demographics.
- Provide actionable recommendations for improving product positioning.
- Develop interactive dashboards for business decision-making.

## ▶️ How to Run the Project

### Tableau

1. Download the dataset.
2. Open Tableau Desktop.
3. Connect the dataset.
4. Open the Tableau Workbook.
5. Explore dashboards and stories.

### Flask Application

Clone the repository:

```bash
git clone https://github.com/your-username/Product-Positioning-Analysis.git
```

Navigate to the project folder:

```bash
cd Product-Positioning-Analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Flask server:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```
